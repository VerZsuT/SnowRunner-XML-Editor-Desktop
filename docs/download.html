v0.5.0 (актуальный)

  • Обновление меню:
    • Все пункты меню теперь переведены
    • Добавлен пункт Файл > Открыть > Дополнения (открыть любой файл дополнений)
    • Добавлен пункт Помощь (позволяет перейти на GitHub, YouTube и на страницу snowrunner.mod.io)
    • Добавлен пункт Файл > Настройки > Язык (менять язык)
    • Добавлен пункт Файл > Настройки > Пути (менять пути)
    • Добавлен пункт Файл > Настройки > Разработчик ("режим разработчика")
  • Переведены ошибки и уведомления
  • Добавлено меню в окне первоначальной настройки (функционал ограничен)
  • Добавлен выбор папки [strings] (для правильного отображения названий в программе)
  • Названия при выбранной папке [strings] теперь отображаются так же как в игре
  • Переводы преобразованы в JSON формат
  • Изменение поведения окон:
    • при закрытии текущего старое показывается
    • при открытии нового старое скрывается
  • Добавлен "режим разработчика"
